Public abstract, class Abstract, and namespace goatee.Action

Abstract action class implementing a basic action

exports.Abstract = class Abstract

Public method process abstract

Derivates must override this method and perform the action.

Parameters:

  • processor must be a goatee.Action.Processor.
    (The calling action)

  • context must be a goatee.Action.Context.
    (The current evaluation context)

  • node must be a Node.
    (The currently processed node)

  • attribute must be a Function.
    (The action's processed value)

Returns a Boolean
(process-execution signal: must stop = true or may continue = false or undefined)
and Can throw an Error
(If this method has not been overridden.)
