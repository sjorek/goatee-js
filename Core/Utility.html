Use return Function.prototype.call.apply(Array.prototype.slice, arguments); instead of the simpler return Array.prototype.slice.call(array, start, opt_end); here because of a bug in the FF and IE implementations of Array.prototype.slice which causes this function to return an empty list if end is not provided.
