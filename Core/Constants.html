Constants

namespace goatee.Core

A class providing static string properties, to avoid subsequent object creation. This raises Internet Explorer ≤ 8 performance and saves memory.

exports.Constants = class Constants

Context-variables

Names of special variables defined by the jstemplate evaluation context. These can be used in js expression in jstemplate attributes.

@VAR_index : '$index' @VAR_count : '$count' @VAR_this : '$this' @VAR_context : '$context' @VAR_top : '$top'

The name of the global variable which holds the value to be returned if context evaluation results in an error. Use Context.setGlobal(Constants.GLOB_default, value) to set this.

@GLOB_default : '$default'

Regular-Expressions

@REGEXP_semicolon : /\s*;\s*/ @REGEXP_trim : /^\s+|\s+$/g @REGEXP_trimLeft : /^\s+/ @REGEXP_trimRight : /\s+$/ @REGEXP_camelize : /-([a-z])/gi @REGEXP_dashify : /(^|[a-zA-Z])([A-Z])/g @REGEXP_isEscaped : /[^\\](\\\\)*$/

JavaScript fragments

Expression evaluation helpers

@STRING_variables : '$_variables' @STRING_data : '$_data' @STRING_with : 'with ($_variables) with ($_data) return '

CSS Properties

These styles are used in by some actions

@CSS_display : 'display' @CSS_position : 'position'

Typeof strings

Constants for possible values of the typeof operator.

@TYPE_boolean : 'boolean' @TYPE_number : 'number' @TYPE_object : 'object' @TYPE_string : 'string' @TYPE_function : 'function' @TYPE_undefined : 'undefined'

Single characters

@CHAR_colon : ':' @CHAR_asterisk : '*' @CHAR_dollar : '$' @CHAR_period : '.' @CHAR_exclamation : '!' @CHAR_ampersand : '&' @CHAR_dash : '-' @CHAR_slash : '/' @CHAR_backslash : '\\' @CHAR_equals : '=' @CHAR_colon : ':' @CHAR_semicolon : ';' @CHAR_doublequote : '"' @CHAR_singlequote : "'"

Whitespace characters

@CHAR_space : " " @CHAR_tab : "\t" # tabulator @CHAR_vtab : "\v" # vertical tabulator @CHAR_cr : "\r" # carriage return @CHAR_lf : "\n" # line feed @CHAR_ff : "\f" # form feed
