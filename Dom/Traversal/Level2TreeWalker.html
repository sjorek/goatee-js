Level2TreeWalker

Public class Level2TreeWalker, extends goatee.Dom.Traversal, namespace goatee.Dom.Traversal, and see TreeWalker Documentation

A class to hold state for a DOM traversal. This implementation depends on DOM Level ≥ 2's TreeWalker.
