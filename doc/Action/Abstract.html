Abstract class implementing a basic action

exports.Abstract = class Abstract ##

Derivates must implement this method to perform the action.

Parameters:

  • action must be a Processor.
    (The calling action)

  • context must be a Context.
    (The current evaluation context)

  • template must be a Node.
    (The currently processed node of the template)

  • attribute must be a Function.
    (Processed value of the content attribute)

Returns a Boolean
