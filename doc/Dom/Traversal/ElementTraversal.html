ElementTraversal

Public class ElementTraversal extends goatee.Dom.Traversal and is in namespace goatee.Dom.Traversal

A class to hold state for a DOM traversal. This implementation depends on the ElementTraversal-interface providing node.firstElementChild and node.nextElementSibling.
